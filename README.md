## 📌 Описание проекта 
Создание анимированной истории по теме Спорта - "Баскетбол"

## 🚀 Что реализовано?
Реализована анимация игры Баскетбол с подсчетом баллов, победой, зрителями.

## 🛠️Используемые технологии?

Использовались основные блоки - Движение, Внешний вид, События, Условия, а также работа с Костюмами спрайтов.

Дополнительно было добавлено Музыкальное сопровождение.
